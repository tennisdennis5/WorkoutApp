# Workout App

Eine ganz simple Workout-App

## Funktionen

- Übungen mit einem Bild und einr kurzen Beschreibung
- Countdown für Übungen
- Soundeffekte
- Fortschrittsbalken
- Speicherung abgeschlossener Workouts mit SharedPreferences

## Verwendete Technologien

- Kotlin
- Jetpack Compose
- ViewModel
- LiveData
- SharedPreferences
